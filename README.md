# AcessoFacilUFSM

Sistema de mapeamento de acessibilidade urbana inspirado em soluções como o Urmob. A ideia é a aplicação permitir a visualização de pontos de interesse com recursos de acessibilidade (como rampas, banheiros adaptados e elevadores) dentro da UFSM, além de possibilitar a avaliação desses locais pelos usuários.
Este projeto tem como objetivo centralizar informações sobre a acessibilidade dentro da UFSM, facilitando a mobilidade e promovendo inclusão por meio da tecnologia.

## Funcionalidades
- Visualização de pontos de acessibilidade no mapa;
- Filtro por tipo (rampa, elevador, banheiro, etc.);
- Exibição de detalhes dos locais;
- Avaliação e comentários de usuários;
- Busca de pontos próximos a localização atual
